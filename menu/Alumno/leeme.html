Explicacion de cada archivo :

1.- disparadores.sql ------ Archivo que guarda los disparadores o triggers que insertar despues de crear la base de datos

2.- proyecto.sql ------ Contiene los 'Create' para poder añadir las tablas y crear la base de datos.

3.- inserciones.sql ------ No es necesario de introducir pero dentro de este archivo se encuenrtran inserciones de ejemplo

Para los script existe un manual en la carpeta 'Manuales' se han diseñado para que la configuracion sea mas sencilla
